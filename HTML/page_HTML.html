Balises:

  • Balises paires:
    <-body> texte <-/body>
  • Balises orphelines:

Les Attributs:

Options des balises, placé après le nom de la balise.
<-balise attribut= « valeur »

Doctype = indique qu’une page est en HTML 5
Balise <-html> = balise principale, englobe le contenu de la page
<-head> et <-body> = Head donne des info sur le titre, l’encodage de la page c’est généralement court. Infos contenues sont pas sur la page
Body partie principale de la page, tout ce qui est écrit est dedans et est affiché.
Balise <-p> : pour le texte et utiliser la balise <-br/> pour sauter une ligne.

Les titres:

<-h1> à <-h6> en ordre d’importance. Pas de h2 sans h1.

Balise <-em>:
Met en valeur un texte (en italique)
Balise <-strong>:
met en valeur un texte (en gras)
Balise <-mark>:
surligne un texte

Les listes:

Liste non ordonné ou puce:
    ul = unordered list.
    Chaque element de liste : <-li>, li = List item.
    Ajouter une autre <-ul> dans <-li> pour du plus complexe
Listes ordonnées ou listes numérotées: <-ol>
ol = ordered list
Liste definition (plus rare) : <-dl>(délimiter liste)
= data list, <-dt>(délimiter terme), et <-dd>(délimiter definition d'un terme)

Créer des liens :

Balise <-a>: attribut href ; valeur de l'attribut est URL de la page.
Ex : <-a href="https//google.com">Google<-/a>
Pour récupérer une page dans un dossier précédent :
href="../page.html"> les 2 points signifient "plus haut" dans l'arborescence.

Lien vers une ancre :

C'est un point de repère à mettre dans une page.
On utilise id="" dans une balise, Et href="#nomdel'id" pour y accéder.

Pour une ancre sur une autre page :

On met le nom de la page puis un # suivit du nom de l'ancre.
Ex: <-a href="page.html#ancre">
Pour afficher une infobulle au survol du lien :
Ajouter Title a la balise.
Pour ouvrir un lien dans un nouvel onglet/fenêtre :
Ajouter l'attribut target="_blank".
Maj+clic = nouvelle fenêtre et ctrl+clic = nouvel onglet.
Pour envoyer un mail :
<-a href="mailto:mail@truc.con">
Télécharger un fichier :
<-a href="fichier.zip">

Les images:

Les images sur internet sont compressées.
Jpg( ou jpeg ), png (portable network graphics) plus récent et puissant les images peuvent etres transparentes: compression plus puissantes et 2 versions(8 bits 256 couleurs) et 24 bits (16 millions), et le gif(image anime ou vidéo loop).
<-img src="image" alt="texte descriptif">

Miniature cliquable:

<-a href="lien image"><-img src="image petite"><-/a>

Figures :

Éléments qui viennent compléter le texte pour illustrer le contenu d'une page.
Balise paire. On peut ajouter une image et une légende (ou caption).
Ex: schéma, images, citations …
<-figure> <-figcaption><-/figcaption> <-img/><-/figure>
